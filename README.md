# House Price Prediction

A Machine Learning project that predicts house prices using an XGBoost Regressor and a Streamlit web application.

## Tech Stack

* Python
* Pandas
* Scikit-Learn
* XGBoost
* Streamlit

## Model Performance

* R² Score: 0.899
* MAE: $18,140

## Run Locally

```bash
git clone https://github.com/MKxFurkan/House-price-prediction.git
cd House-price-prediction

pip install -r requirements.txt
streamlit run app.py
```
